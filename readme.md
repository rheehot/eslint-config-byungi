## Install
```sh
yarn add --dev eslint eslint-config-byungi
```

## Usage
`.eslintrc`
```json
{
    "extends": "byungi"
}
```

### Subsets
- `byungi/old`
- `byungi/react`
- `byungi/hyperapp`
